Claude Code
专线快速接入

高速稳定的 Claude API 中转服务，即开即用，按步骤完成配置。

https://bestaigate.top
1
安装 Git
Windows 环境必装前置依赖
如已安装可跳过
ℹ️ 系统要求：Windows 10+ · 需要 Git for Windows
powershell 
winget install --id Git.Git -e --source winget
2
安装 Claude Code
三种安装方式，任选其一
powershell 
irm https://claude.ai/install.ps1 | iex
cmd 
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
powershell 
winget install Anthropic.ClaudeCode
3
配置 settings.json
设置 API 密钥与专线地址（推荐）
⚠️ 请将 sk-xxxxxx 替换为你在控制台生成的 API 密钥，妥善保管，切勿公开。
📁 配置文件位置：%USERPROFILE%\.claude\settings.json
settings.json 
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-xxxxxx",  // 替换为你的 API Key
    "ANTHROPIC_BASE_URL": "https://bestaigate.top"
  }
}
4
配置 .claude.json
跳过新手引导，直接进入工作模式
📁 配置文件位置：%USERPROFILE%\.claude.json（在已有文件中追加此 key，重启后生效）
.claude.json 
{
  "hasCompletedOnboarding": true
}
5
启动 Claude Code
配置完成，直接运行
terminal 
claude
✅ 配置完成后运行即可通过专线连接，享受高速稳定的 Claude API 服务。

更多公益节点推荐
社区维护的免费/优质 Claude API 中转服务
⚠️ 公益节点稳定性可能波动，建议收藏多个备用。使用前请阅读各节点的使用条款。
🌐
Linux.do 论坛公益节点
社区驱动的免费 Claude API 中转服务，需要论坛账号认证。
Base URL: https://api.linux.do/v1
获取方式: 注册 linux.do 论坛账号，在个人设置中生成 API Key
特点: 免费额度充足 · 社区支持 · 需要论坛活跃度
🚀
Aurora 开源中转项目
开源的 Claude API 代理服务，支持自部署或使用公共实例。
Base URL: https://api.aurora.chat/v1
获取方式: 访问 aurora.chat 注册账号，控制台生成密钥
特点: 开源透明 · 支持多模型 · 可自建部署
⚡
ClaudeAPI.ai 中转服务
稳定的第三方 Claude API 中转，提供免费试用额度。
Base URL: https://api.claudeapi.ai/v1
获取方式: 访问 claudeapi.ai 注册并获取 API Key
特点: 响应速度快 · 新用户免费额度 · 付费套餐可选
💡 配置方法：将上述任一 Base URL 替换到 settings.json 中的 ANTHROPIC_BASE_URL 字段即可。
